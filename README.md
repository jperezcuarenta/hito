# hito 🪡

Decorate GitHub activity with hash-seeded Hitomezashi patterns.

---

## Entry Points

### 1. view
Visualizes the pattern locally.

### 2. weave
Iterates through the last 52 weeks and creates backdated empty commits to paint the existing grid. Run this manually once.

### 3. stitch
A lightweight check that only targets the current date. If the pattern requires a stitch today, it adds 20 commits. Designed for GitHub Actions/Cron.
